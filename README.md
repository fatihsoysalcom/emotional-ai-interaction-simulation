# Emotional AI Interaction Simulation

This example demonstrates a simplified AI that attempts to understand and respond to user emotions, contrasting it with a purely rule-based approach. It simulates the core concept of "SoulForge" by generating empathetic responses based on detected emotional keywords in user input, aiming for a deeper connection.

## Language

`python`

## How to Run

Save the code as `main.py`. Open a terminal, navigate to the directory, and run `python main.py`. Type messages in Turkish and observe the AI's emotional responses.
